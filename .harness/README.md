# Harness: long-term learnings and repo artifacts

This directory stores long-term learnings, session artifacts, and repository-
specific recommendations that agents and contributors can consult before
making persistent changes. Files here are intended to be evergreen notes — do
not include session-specific language or ephemeral transcripts.

Typical contents:

- `tailscale-learnings.md` — operator-specific recommendations and safe defaults
- (other files) — repository lessons, gotchas, and non-actionable observations

When making changes to repository rules or canonical docs, check this folder
first for relevant guidance and add new long-term notes here rather than
committing transient session transcripts.
